### Search Flow Overview

- Intent detection identifies query type (direct match, entity-based, semantic)
- Adaptive vector search with relevance scoring for semantic intent
- Mode decision selects broad (exploratory) or narrow (precision) search
- Result fusion combines exact, graph traversal, and semantic results
- Deduplication and ranking produce the final ordered results
